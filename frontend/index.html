Decentralized Stablecoin UI

Interact with your deployed DSCEngine on Sepolia.

Quick Start

  1. Choose a network preset and paste your DSCEngine address from forge script.
  2. Connect Wallet → Switch Network if needed → Load Contract.
  3. Ensure you hold WETH/WBTC at the preset addresses, then approve and deposit.
  4. Use Refresh Data after each transaction to update your position.
